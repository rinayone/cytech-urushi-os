フレックスボックスの基本

Flexbox（フレキシブルボックス）は、CSSのレイアウトモジュールの一つです。コンテナ内の要素を柔軟に配置・整列するための仕組みで、 従来の float や position によるレイアウトよりも直感的にレスポンシブなデザインを実現できます。

Flexbox を使うには、親要素（Flexコンテナ）に display: flex; を指定します。 子要素（Flexアイテム）は自動的にフレックスアイテムとなり、コンテナの中で柔軟に配置されます。 

<!-- HTML -->
<div class="container">
  <div class="item">Item 1</div>
  <div class="item">Item 2</div>
  <div class="item">Item 3</div>
</div>

<!-- CSS -->
.container {
  display: flex;
  justify-content: space-between;
  align-items: center;
  gap: 16px;
}

.item {
  flex: 1;
  padding: 20px;
  background: #f0f0f0;
  border-radius: 8px;
}

justify-content は主軸方向（デフォルトは横方向）の配置を制御し、align-items は交差軸方向（デフォルトは縦方向）の配置を制御します。
