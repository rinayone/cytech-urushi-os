UI/UX デザイナーの推奨

第一推奨: Ring = A1（シアン→青） + Done = B1（紫 α=0.3）
→ 派手すぎず CTU パレット内で揃う、Today枠（シアン）と明確に区別可能

暖かみが欲しい場合: Ring = A4（ピンク→紫） + Done = B1（紫）
達成感を出したい場合: Ring = A2（紫→青 KANAME） + Done = B3（緑）
クールに統一: Ring = A1 + Done = B2（青）

番号で指定してください（例: 「Ring A1 / Done B1 で」）。本番モックに反映します。
