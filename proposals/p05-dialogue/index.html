P-05 対話モード提案について

背景: v1.1 の P-05 は教科書型レイアウト（左に長文、右にAI・目次）。社内から「学習自体をAIとの対話で進める形にしてはどうか」という案が出ています。

狙い

  • 学習完遂率の向上（最重要KPI）
  • 夜疲れているペルソナ（26歳・非IT営業職）でも続けられる粒度
  • 「家庭教師」コアコンセプトの一段強化

主要なUI/UX変更点

  • 対話メイン + 教材は同期スライドで残す（完全対話化はしない）
  • 営業比喩エンジン: 概念を業務知識に塗り重ねる（漆メタファー）
  • マイクロ進行: 1単元 = 5〜10ターン、30秒チャンク
  • 3つの逃げ道: 「もう一度」「比喩で」「とりあえず次へ」
  • Layer 積み上げ可視化: 上部に塗り重ねが視覚的に増える
  • 客観評価の硬さ維持: レッスン終了時に ✓/△/× で要約

このプロトタイプで体験できること

  • チップを押して 7ターンの対話を進める
  • 右側スライドが対話に同期して切り替わる
  • レッスン終了時に Layer が1枚追加される演出
  • 上部「現行版/提案版」トグルで Before/After 比較

意思決定が必要なこと

  • この方向で v1.2 に進めて良いか
  • 進める場合、教材生成 (AI生成 / 既存コンテンツ再利用) の方針
