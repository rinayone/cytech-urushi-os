v1.1のブラッシュアップは、ユーザーが迷わず・挫折せず・気づいたら学習を完了している体験の実現に集中しました。

P-01 適性診断 → P-02 AI対話 → P-05 学習 → P-06 課題提出
この4画面のフローを徹底的にブラッシュアップしました
Design Principles
2つの設計原則
全ての画面設計は、この2つの原則に基づいています。
🤖
家庭教師AI
AIが一人ひとりに個別カスタマイズされた提案をする。学校の授業ではなく、家庭教師が横にいるような体験。
👆
動線は常に1つ
ユーザーに「次に何をすべきか」を迷わせない。画面上のアクションは基本的に1つだけ。
Core Flow
完遂率を決める4画面
P-01 → P-02 → P-05 → P-06。この流れが途切れなければ、ユーザーは学習を完了します。
P-01
適性診断
KANAMEが挨拶 → 2問の質問 → 「情シスが向いています」＋ロードマップ提示。ネガティブ → ポジティブへの反転。
AI個別提案
P-02
AI対話
「おかえり、前回はここまで進みましたね」→ 復習の問いかけ → 選択肢を選ぶと、気づいたら学習の中にいる。
AI個別提案 動線1つ
P-05
学習実行
1セッション1概念。営業経験に例えた説明。「次へ→」ボタン1つだけ。KANAMEに質問可能。
動線1つ
P-06
課題・テスト
実技課題 → AIが「B1到達です」と根拠付きフィードバック。自信度に応じて次のアクションを1つ提案。
AI個別提案 動線1つ
